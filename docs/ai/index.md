## Project Overview
wthr.lol is a no-bullshit weather application for web and mobile. It allows users to get weather forecasts based on their location.
It backend is written in Go, and frontend uses HTML/CSS/JS.

## Key Commands
- `make run` - Build and run the application
- `make test` - Run tests
- `make check` - Run static analysis
